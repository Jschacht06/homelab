# Scripts

Use the commands below to download and run a script from this directory.

Replace `file-name` with the local filename you want to save, and replace
`script-name` with the script filename in this repository.

```bash
curl -fsSL -o /tmp/file-name https://raw.githubusercontent.com/Jschacht06/homelab/main/scripts/script-name
chmod u+x /tmp/file-name
/tmp/file-name
```
